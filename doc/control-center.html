A handle to a git repository worktree. Declared by plugins/git.toml; `is_a` edges relate it to the filesystem so file verbs (open, reveal, copy-path) reach repos automatically.

Lattice — `is_a` walk

application/vnd.git.repo ⊑ inode/directory ⊑ inode/*

Walking the lattice means polymorphic verbs accepting inode/* (like the files-plugin open) automatically apply to repos — no per-type wiring needed.

Modifiers adverbs · sigils · aliases — the composition layer (the "how" not the "what")

Modifiers shape how verbs apply, not what they do. Adverbs swap an LLM, a search engine, or a tone; sigils are typing shortcuts for subjects; aliases shortcut whole invocations.

Plugins 30 enabled — the package list, with discovery order

Plugins load lowest-to-highest precedence (system → org → user → CWD); later-loaded wins on name collisions for non-polymorphic entries. Verbs and channels merge by (name, accepts) so same-named-different-accepts coexist (the polymorphic dispatch).
